# Pokémon Trainer Journal

A responsive, completionist watch journal for Pokémon animation. It includes episode-level tracking across the major anime eras, feature films, specials and web animation, with ratings, notes, viewing history, statistics, search, filters, and JSON backup/restore.

## Local development

```bash
npm ci
npm run dev
```

## Production validation

```bash
npm run lint
npm run build
```

Progress is saved locally in the browser under `pokemon-journal-progress`.

This is a fan-made project and is not affiliated with Nintendo, Game Freak, Creatures, or The Pokémon Company.
