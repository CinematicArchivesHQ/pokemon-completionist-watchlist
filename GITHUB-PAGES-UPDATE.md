# GitHub Pages update

Copy everything in this update ZIP into the root of the `pokemon-completionist-watchlist` repository and allow it to replace files with the same names.

The ZIP intentionally does not contain `public/assets`; keep the asset folders already uploaded to GitHub.

After committing the update:

1. Open the repository's **Settings**.
2. Select **Pages**.
3. Under **Build and deployment**, set **Source** to **GitHub Actions**.
4. Open the repository's **Actions** tab and wait for **Deploy to GitHub Pages** to finish.
5. Visit `https://cinematicarchiveshq.github.io/pokemon-completionist-watchlist/`.
